   - Contenido: Técnico realizando inspección en instalación industrial
   - Formato: WebP optimizado

2. **recomendacion-estrategica.webp**
   - Dimensiones recomendadas: 1200x900px (ratio 4:3)

3. **implementacion-360.svg** (actualmente se usa el diagrama SVG en su lugar)
